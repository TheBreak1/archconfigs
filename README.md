в .config лежат:

- конфиги openbox+rofi+GTK3 для удобства пользования/красоты

чтобы запихнуть надо:
> git clone https://github.com/TheBreak1/archconfigs.git ; cd archconfigs ; sudo cp .configs ~/

должно сработать после перезапуска
